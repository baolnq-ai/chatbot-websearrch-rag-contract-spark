# Reports

Thư mục này lưu các báo cáo ngắn cho từng task hoặc phiên kiểm thử quan trọng của dự án.

Mỗi file báo cáo nên nêu mục tiêu, phạm vi, thay đổi chính, lệnh verify đã chạy, kết quả, blocker và rủi ro còn lại. Báo cáo trong thư mục này liên kết ngược tới plan trong `plans/` và log trong `logs/` khi task có nhiều phase.

## Báo cáo mới

- `demo-banner-gif-20260527-v1.md`: báo cáo làm demo banner WebP/GIF 36 giây ở 24fps, có đoạn test chat và evidence ảnh.
- `benchmark-10-30-gb10-20260527-v1.md`: benchmark 16 câu hỏi/22 checks trên GB10/DGX Spark local, có upload file, embedding, vLLM, RAG, web search, contract và dashboard ảnh.
