# Kết quả benchmark

Thư mục này chứa kết quả sinh ra từ `scripts/run-benchmark.mjs`.

Các file chính:

- `results.json`: dữ liệu thô đầy đủ.
- `results.csv`: bảng kết quả dễ import.
- `report.md`: báo cáo chữ ngắn gọn.
- `dashboard.html`: dashboard đọc bằng browser/GitHub.
- `dashboard.png`: ảnh chụp dashboard để làm evidence.
